API

GET /api/v1/redis
Return the key/value of the redis store [debugging only].
    Status codes
  • 200 / OK - Status normal
  • 200 / FAIL - Identified problem with service
  • 404 / FAIL - Identified problem with service

Return the status of the identity provider.
    Status codes
  • 200 / OK - Status normal
  • 200 / FAIL - Identified problem with service
  • 404 / FAIL - Identified problem with service
