API

GET /api/v1/bankaccount/<sortcode>
Return the details of a <sortcode> for the bank
    Status codes
  • 200 / OK - Status normal
  • 404 / ERROR - Sortcode not known
GET /api/v1/bankaccount/<sortcode>/<bankaccount>
Return the details of a <sortcode> and <bankaccount> combination
    Status codes
  • 200 / OK - Status normal
  • 404 / ERROR - Sortcode or Bankacount not found
GET /api/v1/identity
Return the status of the identity provider.
    Status codes
  • 200 / OK - Status normal
  • 404 / FAIL - Identity not found
GET /api/v1/catalogue/isbn/
Return details of a monograph by ISBN number
    Status codes
  • 200 / OK - Status normal
  • 404 / FAIL - Monograph not found
